# [15.17.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.17) Multiplicative Operators
- [ ] The type of each of the operands of a multiplicative operator must be a type that is convertible ([§5.1.8](https://docs.oracle.com/javase/specs/jls/se26/html/jls-5.html#jls-5.1.8 "5.1.8. Unboxing Conversion")) to a primitive numeric type, or a compile-time error occurs.
	- [x] primitive
	- [ ] boxed

- [ ] The type of a multiplicative expression is the promoted type of its operands.
