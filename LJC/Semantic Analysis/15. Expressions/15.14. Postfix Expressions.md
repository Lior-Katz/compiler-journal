# [15.14. Postfix Expressions](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.14)

##### [15.14.1.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.14.1) Expression Names

##### [15.14.2.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.14.2) Postfix Increment Operator `++`
- [ ] The result of the postfix expression must be a variable of a type that is convertible ([§5.1.8](https://docs.oracle.com/javase/specs/jls/se26/html/jls-5.html#jls-5.1.8 "5.1.8. Unboxing Conversion")) to a numeric type, or a compile-time error occurs.
	- [x] variable
	- [ ] conversion
		- [x] numeric
		- [ ] unboxing conversion
- [x] The type of the postfix increment expression is the type of the variable. The result of the postfix increment expression is not a variable, but a value.
- [ ] A variable that is declared `final` cannot be incremented

##### [15.14.3.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.14.3) Postfix Decrement Operator `--`
- [ ] The result of the postfix expression must be a variable of a type that is convertible ([§5.1.8](https://docs.oracle.com/javase/specs/jls/se26/html/jls-5.html#jls-5.1.8 "5.1.8. Unboxing Conversion")) to a numeric type, or a compile-time error occurs.
	- [x] variable
	- [ ] conversion
		- [x] numeric
		- [ ] unboxing conversion
- [x] The type of the postfix decrement expression is the type of the variable. The result of the postfix decrement expression is not a variable, but a value.
- [ ] A variable that is declared `final` cannot be decremented
