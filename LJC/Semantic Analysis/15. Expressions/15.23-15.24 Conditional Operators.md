##### [15.23.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.23) Conditional-And Operator `&&`
- [ ] Each operand of the conditional-and operator must be of type `boolean` or `Boolean`, or a compile-time error occurs.
	- [x] primitive
	- [ ] boxed
- [x] The type of a conditional-and expression is always `boolean`.

##### [15.24.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.24) Conditional-Or Operator `||`
- [ ] Each operand of the conditional-or operator must be of type `boolean` or `Boolean`, or a compile-time error occurs.
	- [x] primitive
	- [ ] boxed
- [x] The type of a conditional-or expression is always `boolean`.
