# [15.15. Unary Operators](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.15)
##### [15.15.1.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.15.1) Prefix Increment Operator `++`
- [ ] The result of the unary expression must be a variable of a type that is convertible ([§5.1.8](https://docs.oracle.com/javase/specs/jls/se26/html/jls-5.html#jls-5.1.8 "5.1.8. Unboxing Conversion")) to a numeric type, or a compile-time error occurs.
	- [x] variable
	- [ ] conversion
		- [x] numeric
		- [ ] unboxing conversion
- [x] The type of the prefix increment expression is the type of the variable. The result of the prefix increment expression is not a variable, but a value.
- [ ] A variable that is declared `final` cannot be incremented
##### [15.15.2.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.15.2) Prefix Decrement Operator `--`
- [ ] The result of the unary expression must be a variable of a type that is convertible ([§5.1.8](https://docs.oracle.com/javase/specs/jls/se26/html/jls-5.html#jls-5.1.8 "5.1.8. Unboxing Conversion")) to a numeric type, or a compile-time error occurs.
	- [x] variable
	- [ ] conversion
		- [x] numeric
		- [ ] unboxing conversion
- [x] The type of the prefix decrement expression is the type of the variable. The result of the prefix decrement expression is not a variable, but a value.
- [ ] A variable that is declared `final` cannot be decremented

##### [15.15.3.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.15.3) Unary Plus Operator `+`
- [ ] The type of the operand expression of the unary `+` operator must be a type that is convertible ([§5.1.8](https://docs.oracle.com/javase/specs/jls/se26/html/jls-5.html#jls-5.1.8 "5.1.8. Unboxing Conversion")) to a primitive numeric type, or a compile-time error occurs.
	- [x] numeric
	- [ ] unboxing conversion
- [ ] The type of the unary plus expression is the promoted type of the operand. 
- [x] The result of the unary plus expression is not a variable, but a value, even if the result of the operand expression is a variable.

##### [15.15.4.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.15.4) Unary Minus Operator `-`
- [ ] The type of the operand expression of the unary `-` operator must be a type that is convertible ([§5.1.8](https://docs.oracle.com/javase/specs/jls/se26/html/jls-5.html#jls-5.1.8 "5.1.8. Unboxing Conversion")) to a primitive numeric type, or a compile-time error occurs.
	- [x] numeric
	- [ ] unboxing conversion
- [ ] The type of the unary minus expression is the promoted type of the operand.
- [x] The result is a value

##### [15.15.5.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.15.5) Bitwise Complement Operator `~`
- [ ] The type of the operand expression of the unary `~` operator must be a type that is convertible ([§5.1.8](https://docs.oracle.com/javase/specs/jls/se26/html/jls-5.html#jls-5.1.8 "5.1.8. Unboxing Conversion")) to a primitive integral type, or a compile-time error occurs.
	- [x] numeric
	- [ ] unboxing conversion
- [ ] The type of the unary bitwise complement expression is the promoted type of the operand.
- [x] result is a Value

##### [15.15.6.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.15.6) Logical Complement Operator `!`
- [ ] The type of the operand expression of the unary `!` operator must be `boolean` or `Boolean`, or a compile-time error occurs.
	- [x] primitve
	- [ ] unboxing
- [x] The type of the unary logical complement expression is `boolean`.
- [x] result is a Value
