# [15.19.](https://docs.oracle.com/javase/specs/jls/se26/html/jls-15.html#jls-15.19) Shift Operators
- [x] It is a compile-time error if the type of each of the operands of a shift operator, after unary numeric promotion, is not a primitive integral type.
- [x] The type of the shift expression is the promoted type of the left-hand operand.
